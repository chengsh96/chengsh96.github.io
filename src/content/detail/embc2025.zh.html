基于智能手表的手势控制与振动反馈用于多活动假肢控制（EMBC 2025）

科研
IEEE EMBC 2025 人在环控制 可穿戴交互界面 振动触觉反馈 假肢行走控制
IEEE 论文 PDF

本研究提出了一种 基于智能手表的人–机交互接口， 使动力膝–踝假肢使用者能够显式地下达行走模式切换指令， 并即时获得 振动触觉与视觉反馈。 该系统被设计为自动意图识别的实用补充与人工覆盖手段， 有效提升了真实使用场景中的透明性、信任度与可靠性。

本人贡献

  • 设计并实现智能手表 UI， 支持四种活动模式（步行、上楼、下楼、坐立）， 并将每种模式映射为直观的滑动手势。
  • 为不同活动模式设计差异化的振动触觉反馈， 实现无需注视即可确认控制器状态。
  • 基于 TCP 网络实现智能手表与假肢控制器之间的可靠双向通信。
  • 将交互系统集成至运行于实时 LabVIEW 平台的动力膝–踝假肢控制器中。

研究意义

  • 通过低认知负担的交互方式， 保留并强化用户对行走模式切换的主导权。
  • 在自动意图识别不确定或被关闭时， 仍可保证系统的可靠运行。
  • 在数百次活动转换实验中验证了高可靠性， 具备低延迟、零误触发的实际性能。
  • 为“自动控制 + 用户显式指令”相结合的部署路径提供了可行方案。

技术要点

人机交互（HRI） 可穿戴 UI / UX 振动触觉反馈设计 Android / Java TCP 网络通信 实时假肢控制 用户实验与验证

本研究于密歇根大学 LocoLab 开展，导师为 Robert D. Gregg 教授。
