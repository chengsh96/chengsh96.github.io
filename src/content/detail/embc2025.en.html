Smartwatch Gesture & Vibrotactile Feedback for Multi-Activity Prosthesis Control (EMBC 2025)

Research
IEEE EMBC 2025 Human-in-the-Loop Control Wearable UI Vibrotactile Feedback Prosthetic Locomotion
IEEE Paper PDF

This work presents a smartwatch-based human–robot interface that allows users of a powered knee-ankle prosthesis to explicitly command locomotion mode transitions while receiving immediate vibrotactile and visual feedback. The system is designed as a practical override and complement to automatic intent recognition, improving transparency, trust, and reliability in real-world use.

What I contributed

  • A smartwatch UI that presents four activity modes (walk, stair ascent, stair descent, sit-stand), with each mode mapped to an intuitive swipe gesture.
  • Distinct vibrotactile patterns associated with each mode, enabling eyes-free confirmation of controller state.
  • Reliable bidirectional communication between the smartwatch and prosthesis controller using TCP networking.
  • Integration with a powered knee-ankle prosthesis controller running on a real-time LabVIEW system.

Why it matters

  • Preserves user agency by providing a clear, low-cognitive-load method for commanding transitions.
  • Enables reliable operation even when automatic intent recognition is uncertain or disabled.
  • Demonstrates high reliability across hundreds of transitions with minimal latency and no false commands.
  • Provides a deployment-ready pathway for combining autonomy with explicit user control.
