Transfer Learning for Efficient Intent Prediction (RA-L 2024)

Research
IEEE Robotics & Automation Letters Transfer Learning Intent Prediction Data-Limited Training
RA-L 2024 Paper

This project develops a transfer learning pipeline to improve locomotion intent prediction for new transfemoral amputee users when only a small amount of labeled data is available. Instead of training a fully subject-dependent model from scratch (which requires extensive per-user data), we fine-tune pre-trained CNN models to quickly adapt to a novel user while retaining useful feature representations learned from prior datasets.

What I contributed

  • A CNN-based intent classifier using a Fully Convolutional Network backbone, designed for continuous-time prediction.
  • Transfer learning strategy: freeze early convolutional blocks and fine-tune remaining layers for a new amputee user, improving accuracy with limited training circuits.
  • Comparative evaluation across subject-dependent, subject-independent, and transfer-learning models, plus sensor ablations.

Key findings (why it matters)

  • Transfer learning achieves error rates comparable to heavier subject-dependent training, but with substantially less user data.
  • Pre-training can leverage both amputee and able-bodied datasets; performance improves as more subject data is available.
  • A thigh IMU + load cell is identified as a practical, efficient sensing choice for deployment-oriented systems.
