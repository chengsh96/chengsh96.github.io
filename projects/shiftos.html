ShiftOS is the production control system powering the Moonwalkers robotic footwear. I led the design and deployment of real-time locomotion behaviors across ShiftOS 3.0 and 3.1, focusing on balance robustness, low-latency transitions, and safety-critical operation in unconstrained real-world environments.

The system integrates onboard sensing, state estimation, and user-intent inference to adapt assistance across walking modes, terrain conditions, and user profiles. Emphasis was placed on deterministic timing, fault tolerance, and deployability on resource-constrained embedded hardware.

  • Balance-aware locomotion control: Designed control behaviors that adapt assistance based on user posture and ground interaction, improving stability on uneven terrain and during speed changes.
  • Low-latency state transitions: Implemented hybrid mode switching with low latency, enabling smooth transitions between walking modes while preserving user intent and comfort.
  • Safety-critical constraints: Integrated real-time safety checks and fallback behaviors to handle disturbances, sensor anomalies, and edge cases without disrupting user motion.
  • Embedded systems & RTOS integration: Deployed the full control stack on an RTOS-based embedded platform, emphasizing deterministic scheduling, memory safety, and long-term reliability in daily use.
  • Production validation: Contributed to internal testing, field trials, and performance validation supporting the public releases of ShiftOS 3.0 and 3.1.
